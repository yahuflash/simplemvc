SimpleMVC是什么？
===============
一个轻量级的，可扩展的，工具化自动化的，具有（或将具有）丰富的易于使用的模块功能（通过扩展机制扩展）的ActionScript 3.0框架。
支持以其开发具有MVCS结构的Flex、纯AS3及AIR桌面或移用项目。

SimpleMVC框架代码设计细则：
======================
1. 所有方法，如果是public公开的，则首字母大写
2. 所有类属性，如果是public公开的，则首字母大写，如果是常量全部大写，以下线线分隔
3. 函数参数首字母小写，私有类方法及属性首字母小写
